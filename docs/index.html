Cómo interpretar estos datos

La observación corresponde a la estación meteorológica operativa asociada a la localidad. El pronóstico puede estar compartido entre localidades cercanas. Cuando el SMN no publica un pronóstico actual para una base antártica, se muestra el último pronóstico oficial disponible con su fecha original y una advertencia visible.
